# VeraCrypt Password Decoding and File Access

## Objective
The objective of this task is to decode an encrypted password
and use it to unlock an encrypted file using VeraCrypt.

## Tool Used
- VeraCrypt
- Hash decoding tools
- Kali Linux

## Task Description
In this task, an encrypted file is provided along with a password
stored in hash format. The password is first decoded and then used
to unlock the encrypted VeraCrypt file.

After unlocking the file, a secret code is found inside it.

## Steps Performed
1. Opened the encoded.txt file containing the hashed password.
2. Identified the hash type.
3. Decoded the hash using appropriate tools.
4. Installed and opened VeraCrypt.
5. Entered the decoded password to unlock the file.
6. Accessed the file and found the secret code.
7. Took screenshots of the process.

## Output
The encrypted file was successfully unlocked and
the secret code was retrieved.
Screenshots and files are included in this repository.

## Learning Outcome
- Learned how encryption and hashing work.
- Understood basic password decoding techniques.
- Gained hands-on experience with VeraCrypt.

and on provided files.
